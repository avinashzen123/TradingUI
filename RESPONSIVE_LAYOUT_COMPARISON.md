## Key Differences Summary

| Element | Desktop | Tablet | Mobile |
|---------|---------|--------|--------|
| **Layout** | Horizontal | Mixed | Vertical |
| **Sidebar** | Left (300px) | Top (40vh) | Top (40vh) |
| **Header** | Horizontal | Vertical | Vertical |
| **Selectors** | Side-by-side | Stacked | Stacked |
| **Indicators** | Wrap horizontal | Vertical | Vertical |
| **Chart Height** | 400px | 300px | 250px |
| **Indicator Charts** | 150px | 150px | 120px |
| **Grid Columns** | Auto-fit | 2 columns | 1 column |
| **Modal Width** | Max 800px | Full - 20px | Full - 10px |
| **Touch Targets** | 32px | 44px | 44px |
| **Font Sizes** | Base | Slightly smaller | Optimized |
| **Padding** | Standard | Reduced | Minimal |

## Responsive Behavior

### Width Breakpoints
- **>768px**: Desktop layout
- **≤768px**: Tablet layout (vertical stacking begins)
- **≤480px**: Mobile layout (further optimizations)

### Height Considerations
- **Landscape**: Reduced chart heights to fit screen
- **Portrait**: Standard mobile heights

### Orientation Changes
- **Portrait → Landscape**: Charts shrink, sidebar reduces
- **Landscape → Portrait**: Charts expand, sidebar expands

## Visual Hierarchy

### Desktop
1. Header (horizontal)
2. Indicator Manager (horizontal)
3. Main Chart (large)
4. Indicator Charts (medium)
5. Analysis Results (grid)

### Mobile
1. Header (vertical stack)
2. Indicator Manager (vertical stack)
3. Main Chart (optimized)
4. Indicator Charts (compact)
5. Analysis Results (single column)

## Interaction Patterns

### Desktop
- Hover effects
- Precise mouse clicks
- Keyboard navigation
- Multi-column layouts

### Mobile
- Touch targets (44x44px)
- Tap interactions
- Swipe scrolling
- Single column layouts
- No hover effects
- Active state feedback

## Performance Considerations

### Desktop
- Larger charts (more pixels)
- More simultaneous elements
- Hover animations

### Mobile
- Smaller charts (fewer pixels)
- Stacked elements (less simultaneous rendering)
- Touch feedback only
- Optimized for 60fps scrolling
